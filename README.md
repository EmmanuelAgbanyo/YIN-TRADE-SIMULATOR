# YIN Trade Simulator

**YIN Trade Simulator** is a dynamic, client-side stock trading game that allows you to experience the thrill of the market without any financial risk. Built with a modern tech stack, this simulator provides a feature-rich environment where you can create a profile, manage a virtual portfolio, and even team up with friends to compete for the top spot. The application runs entirely in your browser, making it easy to get started.

## Key Features

- **Profile Management**: Create and manage your own trading profile. Secure it with a password for added privacy.
- **Team Collaboration**: Form trading teams with friends, view invite codes, and compete together.
- **Portfolio Tracking**: Monitor your cash, portfolio value, and active orders in real-time.
- **Dynamic Market**: Experience a simulated market with fluctuating stock prices and market sentiment.
- **AI-Powered Chatbot**: Get trading insights and assistance from an integrated AI chatbot powered by the Gemini API.
- **Onboarding and Guidance**: A comprehensive onboarding tour and guide modal to help you get started.

## Tech Stack

- **Framework**: [React](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [TailwindCSS](https://tailwindcss.com/)
- **AI**: [Google Gemini API](https://ai.google.dev/)

## Getting Started

Follow these steps to run the YIN Trade Simulator locally.

### Prerequisites

- [Node.js](https://nodejs.org/) installed on your machine.

### Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/2yin-trade-simulator.git
    cd 2yin-trade-simulator
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Set up your environment variables:**
    Create a `.env.local` file in the root of your project and add your Gemini API key:
    ```
    VITE_GEMINI_API_KEY=your_gemini_api_key_here
    ```

4.  **Run the application:**
    ```bash
    npm run dev
    ```
    The application will be available at `http://localhost:5173`.

## How to Play

1.  **Create a Profile**: Start by creating a unique trading profile.
2.  **Explore the Dashboard**: Familiarize yourself with the dashboard, where you can view market data, your portfolio, and order history.
3.  **Buy and Sell Stocks**: Place buy and sell orders for various stocks based on market trends.
4.  **Form a Team**: Create a team and invite your friends to trade together.
5.  **Use the Chatbot**: Interact with the AI chatbot for tips and information.
